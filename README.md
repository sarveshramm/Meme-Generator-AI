##  Project Description  

**Meme-Generator-AI** is a fast, interactive, and responsive meme creation web application built using **React + Vite**.  
It allows users to **generate memes instantly**, add **top and bottom text**, and render them with a clean, modern UI and proper meme-style typography.

This project showcases front-end development skills such as **React components**, **hooks**, **form handling**, and **UI layout** with custom CSS.

---

##  Features  

-  **Modern Gradient Header UI**  
-  **Top & Bottom Text Inputs** for classic meme-style captions  
-  **Instant Meme Rendering** on button click  
-  **Responsive Image + Text Overlay** with proper positioning  
-  **Impact-like Meme Font & Text Shadow** for authentic meme aesthetics  
-  Built with **React 19 + Vite 7**  
-  Easy to **deploy on any static hosting / cloud platform**
